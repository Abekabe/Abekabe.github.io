LXC佈署

這是很久以前做NA作業時候留下來的LXC佈署筆記，如果有時間會回來詳細的補完（雖然發現Docker真的好用ww

sudo pacman -S lxc

vim /etc/default/lxc

vim /etc/default/lxc-net

Create LXC

lxc-create -n playtime -t download – –dist archlinux –release current –arch amd64
lxc-start -n playtime
lxc-attach -n playtime
